# QR Code Generator

Générateur de QR codes moderne construit avec Next.js 16 et React 19.

## Site officiel

Accéder à l'application en ligne: [qrcode.arthurp.fr](https://qrcode.arthurp.fr)

## Fonctionnalités

- Génération de QR codes pour texte, URL, email, téléphone, SMS et Wi-Fi
- Personnalisation (couleurs, tailles, styles de modules, coins arrondis, logo)
- Export PNG et SVG
- Historique local des QR codes récents
- Génération en masse via import CSV

## Démarrage local

```bash
npm install
npm run dev
```

Application disponible sur http://localhost:3000

## Vérifications qualité

```bash
npm run lint
npm run build
```

## Stack technique

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS
